# 提案ソリューション

## システム構成

提案するAI基盤は、以下の3層アーキテクチャで構成する。

### アプリケーション層

ユーザーが直接操作するインターフェースとして、Webチャット画面とAPI連携機能を提供する。既存の社内ポータルとシングルサインオン（SSO）で統合し、追加のアカウント管理は不要とする。

### AI推論層

大規模言語モデル（LLM）をオンプレミスGPUサーバーで運用する。外部クラウドAPIへの依存を排除し、機密情報の社外流出リスクをゼロにする。

### データ層

社内文書をベクトル化して格納するベクトルデータベースと、アクセス権限管理システムで構成する。既存のファイルサーバーおよびSharePointからの自動取り込みパイプラインを構築する。

## 導入効果

| 項目 | 現状 | 導入後（見込み） |
|------|------|-----------------|
| 報告書作成時間 | 4時間/件 | 2時間/件 |
| 情報検索時間 | 30分/回 | 5分/回 |
| 定型業務自動化率 | 15% | 70% |

## セキュリティ対策

全通信はTLS 1.3で暗号化し、データはAES-256で保存時暗号化を実施する。アクセスログは365日保持し、不正アクセスの検知にはSIEMとの連携を行う。
