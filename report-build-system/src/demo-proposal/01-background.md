# 背景と課題

## 現状の課題

当社の業務プロセスにおいて、以下の非効率が恒常化している。

- **文書作成業務の属人化** — レポート作成に1件あたり平均4時間を要し、フォーマットの統一が困難
- **情報検索の非効率性** — 社内ナレッジが分散しており、必要な情報の検索に平均30分以上を消費
- **定型業務の手動処理** — 月次集計、報告書作成など、自動化可能な業務が手動で行われている

## 市場動向

生成AI技術の急速な進歩により、企業での活用事例が増加している。特にRAG（Retrieval-Augmented Generation）技術の成熟により、社内データを活用した高精度な情報検索と文書生成が実用化段階に入っている。

## プロジェクトの目的

本プロジェクトでは、生成AI基盤を社内に構築し、以下の目標を達成する。

1. 文書作成業務の工数を50%削減
2. 社内情報検索の所要時間を80%短縮
3. 定型業務の自動化率を70%以上に引き上げ
